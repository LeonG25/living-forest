The Living Forest — how the pages connect, and what's next

This updates the original page-map to match what's actually live, and reconciles the two deviations we chose to keep: Home is now “the Sky” (the heavens seen from within) and the Map is “the Globe” (the earth seen from outside, with a year wheel); and the always-present top bar is, for now, a compact ⊕ “lenses” menu on every screen. Two rules still keep anything from hanging inaccessible: the menu is everywhere, and the three node types — Person, Place, Moment — all cross-link. Colour shows status: what's live, what Claude Design has drawn but isn't built, and what still needs design.

The logical order, in phases

Phase 1 · Light up what's already designed

No new Claude Design needed — port existing reskin designs onto real data. Build 1–4: Timeline → Journal → unify the skin (Add/Review/Places/Profile + old games in the new look) → Where Was This? on the globe.

Phase 2 · The node web (needs design)

Design 1–4 → Build 5–8: Person page v2 (“light points”), then the Place, Moment, and Memory/Story pages — the cross-linking that makes the forest feel whole.

Phase 3 · Connective & global (needs design)

Design 5–7 → Build 9–11: Thread-back-to-You, Search, and the Tree lens (with “Who they knew”).

Phase 4 · Deepen Tend + harder games

Design 8 → Build 12–15: Contribute hub, Propose, Record-a-voice, Gaps; then the games that need a data/engine change — Who Is Who? (clue generator), Tangled Thread (places-lived), Missing Voice (narrator field).

Phase 5 · Payoffs & curation

Design 9–12 → Build 16–19: Story Reel / Memory Lane, Manage curators, Themed threads / Connection-found, and What Happened Next? (the one bridge from Wander into Tend).

Phase 6 · Parked big ideas (later)

The globe placement game (a year + names → drop them on the map), the world & family events layer on the globe, and the escape-room quest chain. Each needs its own design pass when we get there.

No-orphan checklist (still holds)

  • Every node page (Person, Place, Moment) links to the other two — no dead-end nodes.
  • The ⊕ menu is on every screen, so lenses, Journal, Contribute and Profile are always one tap away.
  • Hardware Back closes an open card first; otherwise steps one real screen back; only the Sky exits, after a confirm.
  • Challenges return somewhere (Reveal → Person) and write to the Journal.
  • Contribute flows to the Review queue; approved items publish back into the web.
  • “What Happened Next?” is the one bridge from Wander into Tend (a gap → Record).
  • Every lens reaches the others, and all open the same nodes — one web, several doors.
Deviations from the original plan (kept on purpose): Home became the Sky (heavens from within) and the Map became the Globe (earth from outside) — a cleaner split of the two lenses; the year wheel lives on the globe (time-on-map, not a replacement for the Timeline lens); the fixed top bar is, for now, the ⊕ menu. Superseded and due for retirement on consolidation: preview-globe.html, the old Leaflet Places map, and the old 2-D “Forest” home tab.

Next step: with this map and roadmap fixed, we write the Claude Design prompts in Design # order — starting with the Person page v2 (Design #1), then Place, Moment, and Memory/Story — while Phase 1 (Timeline, Journal, skin unification, Where Was This?) proceeds in parallel since it needs no new design.
